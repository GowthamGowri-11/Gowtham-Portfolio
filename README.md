# Gowtham M - Professional Portfolio

A modern, professional React portfolio showcasing skills, projects, and achievements.

## Features

- **Smooth Animations**: Framer Motion for professional animations
- **Responsive Design**: Works seamlessly on all devices
- **Modern UI**: Clean, professional design with refined color palette
- **Interactive Components**: Engaging user experience
- **Performance Optimized**: Fast loading and smooth interactions

## Tech Stack

- React 18
- Framer Motion (animations)
- React Icons
- React Intersection Observer
- CSS3 with custom properties

## Installation

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Color Palette

- Primary: #2C3E50 (Dark Blue-Gray)
- Secondary: #34495E (Slate)
- Accent: #E74C3C (Professional Red)
- Background: #FFFFFF, #F8F9FA, #ECF0F1
- Text: #2C3E50, #7F8C8D

## Customization

Update personal information in the component files:
- Contact details in `Hero.js` and `Contact.js`
- Projects in `Projects.js`
- Skills in `Skills.js`
- Experience in `Experience.js`
